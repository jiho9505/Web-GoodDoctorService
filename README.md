## 프로젝트 개발내용
<p>- 프론트는 REACT 이용 / 백은 EXPRESS 이용 / DB는 MONGO DB 이용</p> 
<p>- CSR 방식</p>  

## 프로젝트 개발동기
<p>- 환자들이 명의를 선택할 권리를 넓히기 위해 만들었다. </p>
<p>- 명의 기준은 EBS 명의 출연 혹은 네이버 회원수 많은 카페 기준이 되었고 </p>
<p>- 카페 공식 추천 명의 리스트를 이용하였다. </p>
<p>- 명의는 관절,인대,힘줄 관련된 명의들로 구성하였다.</p>
